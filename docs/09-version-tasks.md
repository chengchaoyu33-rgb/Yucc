## v1.0.0 - MVP 核心版本

### 版本目标

完成 WMS 系统核心功能，实现基本的入库、出库、库存管理能力。

### 功能范围

| 模块 | 功能点 |
|------|--------|
| 系统管理 | 用户登录/登出、用户管理、角色管理、权限控制 |
| 基础数据 | 商品管理、商品分类、仓库管理、库位管理 |
| 入库管理 | 入库单创建、审核、确认、查询 |
| 出库管理 | 出库单创建、审核、确认、查询 |
| 库存管理 | 实时库存查询、库存流水查询 |
| 首页看板 | 数据统计展示 |
