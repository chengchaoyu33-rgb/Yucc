# WMS-Lite 项目概述

## 一、项目简介

### 1.1 项目名称
**WMS-Lite** - 轻量级仓库管理系统

### 1.2 项目定位
WMS-Lite 是一款面向中小型企业的轻量级仓库管理系统，具有以下核心特点：

| 特点 | 说明 |
|------|------|
| **生产就绪** | MySQL 默认数据库，稳定可靠，适合生产环境 |
| **开箱即用** | 内置演示数据，一键启动即可使用 |
| **功能完整** | 覆盖入库、出库、库存、盘点等核心业务场景 |
| **易于扩展** | 模块化设计，支持多数据库切换（MySQL/PostgreSQL/SQLite） |
| **技术成熟** | Vue 3 + Node.js，社区活跃，文档丰富 |

### 1.3 目标用户
- 中小型电商仓库
- 小型制造企业原料仓
- 第三方物流小型仓库
- 零售门店后仓
- 个人创业者仓库管理

## 二、项目背景

### 2.1 市场痛点

传统 WMS 系统存在以下问题：

```
┌─────────────────────────────────────────────────────────────────┐
│                      传统 WMS 系统痛点                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ❌ 部署复杂     需要专业IT人员，服务器配置繁琐                   │
│  ❌ 成本高昂     软件授权费、实施费、维护费动辄数万               │
│  ❌ 功能冗余     大量中小企业用不到的高级功能                     │
│  ❌ 学习成本高   界面复杂，培训周期长                            │
│  ❌ 定制困难     二次开发门槛高，响应周期长                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 解决方案

WMS-Lite 针对上述痛点提供解决方案：

```
┌─────────────────────────────────────────────────────────────────┐
│                      WMS-Lite 解决方案                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ✅ 部署简单     Docker 一键部署，或双击启动可执行文件            │
│  ✅ 零成本       开源免费，无授权费用                            │
│  ✅ 功能精简     聚焦核心业务，去除冗余功能                       │
│  ✅ 易于上手     界面简洁直观，无需专业培训                       │
│  ✅ 方便定制     代码结构清晰，注释完善，易于二次开发              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 六、快速开始

### 6.1 环境要求

| 环境 | 最低版本 | 推荐版本 |
|------|----------|----------|
| Node.js | 18.0 | 20.x LTS |
| npm | 9.0 | 10.x |
| MySQL | 5.7 | 8.0 |
| Docker (可选) | 20.0 | 24.x |

### 6.2 安装步骤

#### 方式一：源码运行

```bash
# 1. 克隆项目
git clone https://github.com/your-org/wms-lite.git
cd wms-lite

# 2. 安装依赖
npm install

# 3. 配置数据库连接
cp .env.example .env
# 编辑 .env 文件，配置 MySQL 连接信息
# DATABASE_URL="mysql://user:password@localhost:3306/wms_lite"

# 4. 初始化数据库
npm run db:init

# 5. 启动开发服务器
npm run dev
```

#### 方式二：Docker 部署（可选）

> Docker 部署可作为后续扩展方案，当前推荐使用源码部署。

```bash
# 1. 克隆项目
git clone https://github.com/your-org/wms-lite.git
cd wms-lite

# 2. 启动服务
docker-compose up -d

# 3. 查看日志
docker-compose logs -f
```

### 6.3 访问系统

启动成功后，访问以下地址：

| 地址 | 说明 |
|------|------|
| http://localhost:3000 | 前端页面 |
| http://localhost:3001/api | API 接口 |
| http://localhost:3001/api/docs | API 文档 |
