```

│                        性能要求                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1. 响应时间标准                                                │
│  ┌────────────────────────────────────────────────────────────┐│

│  ├────────────────────────────────────────────────────────────┤│

│  │ 页面切换          < 100ms         200ms                     ││

│                                                                 │
│  2. 动画性能优化                                                │
│  ┌────────────────────────────────────────────────────────────┐│

│                                                                 │

│  ┌────────────────────────────────────────────────────────────┐│

│  │                                                            ││

│  │  }                                                         ││
│  │                                                            ││
│  │  /* 全屏切换动画 */                                         ││

│  │  }                                                         ││
│  │                                                            ││
│  └────────────────────────────────────────────────────────────┘│
│                                                                 │

│  │ • 首次内容渲染 (FCP): < 1s                                 ││
│  │ • 最大内容渲染 (LCP): < 2.5s                               ││
```

```
├─────────────────────────────────────────────────────────────────┤

│  │                                                            ││
│  │  平板端 (768px - 1199px):                                  ││
│  │  ┌──────────────────────────────────────────────────────┐ ││
│  │  │ • 减少列数，保持核心内容                               │ ││
│  │  │ • 表格部分列折叠                                       │ ││
│  │  │ • 侧滑弹窗自适应宽度                                   │ ││

│  │                                                            ││
│  │  移动端 (< 768px):                                         ││
│  │  ┌──────────────────────────────────────────────────────┐ ││
│  │  │ • 单列布局，内容优先                                   │ ││
│  │  │ • 表格转为卡片列表                                     │ ││
│  │  │ • 侧滑弹窗全屏展示                                     │ ││

│  │                                                            ││
│  └────────────────────────────────────────────────────────────┘│
```
